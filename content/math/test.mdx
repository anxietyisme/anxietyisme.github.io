## 3. Academic Boxes

You have several custom React components to format mathematical and academic structures. Use them like HTML tags!

### Theorem
```mdx
<Theorem title="Pythagorean Theorem">
  In a right-angled triangle, the square of the hypotenuse side is equal to the sum of squares of the other two sides: $a^2 + b^2 = c^2$.
</Theorem>
```
<Theorem title="Pythagorean Theorem">
  In a right-angled triangle, the square of the hypotenuse side is equal to the sum of squares of the other two sides: $a^2 + b^2 = c^2$.
</Theorem>

### Definition
```mdx
<Definition title="Topological Space">
  A topological space is an ordered pair $(X, \tau)$ where $X$ is a set and $\tau$ is a collection of subsets of $X$ satisfying certain axioms.
</Definition>
```
<Definition title="Topological Space">
  A topological space is an ordered pair $(X, \tau)$ where $X$ is a set and $\tau$ is a collection of subsets of $X$ satisfying certain axioms.
</Definition>

### Proof
```mdx
<Proof>
  Assume $a^2$ is even. Then $a^2 = 2k$ for some integer $k$. 
  By contradiction, if $a$ were odd, $a = 2m + 1$, then $a^2 = 4m^2 + 4m + 1 = 2(2m^2 + 2m) + 1$, which is odd. 
  Thus $a$ must be even.
</Proof>
```

*Other boxes available: `<Lemma>`, `<Remark>`, and `<Example>`.*

## 5. Mathematical Graphs (Mafs)

Plot beautiful, interactive 2D math graphs.

```mdx
<MathGraph y="sin(x) * exp(-0.1 * x)" domain={[-10, 10]} />
```
<MathGraph y="sin(x) * exp(-0.1 * x)" domain={[-10, 10]} />

You can also plot along the Y-axis:
```mdx
<MathGraph x="y * y" domain={[-5, 5]} height={250} />
```
<MathGraph x="y * y" domain={[-5, 5]} height={250} />

## 6. Diagrams & Flowcharts (Mermaid.js)

## 7. Data Charts (Recharts)

Plot arrays of data into line, bar, or area charts.

```mdx
<DataChart 
  type="bar" 
  xKey="name" 
  yKey="score" 
  data={[
    { name: "Math", score: 95 },
    { name: "Physics", score: 88 },
    { name: "CS", score: 100 }
  ]} 
/>
```

## 8. Handwritten Sketches

If you have scanned sketches or images, you can display them with a caption. Just place the image in your `public` folder.

```mdx
<HandwrittenSketch 
  src="/some-image.png" 
  alt="My diagram" 
  caption="Figure 1: Diagram drawn on paper." 
/>
```
