## Motivation

Topology is the study of properties preserved under continuous deformations. Unlike metric spaces, topological spaces strip away the notion of distance and retain only the structure of "nearness" through open sets.

> "A topologist is someone who can't tell the difference between a coffee mug and a donut."

## Definition: Topological Space

A **topological space** is a pair $(X, \tau)$ where $X$ is a set and $\tau \subseteq \mathcal{P}(X)$ is a collection of subsets satisfying:

1. $\emptyset \in \tau$ and $X \in \tau$
2. $\tau$ is closed under arbitrary unions: if $\{U_\alpha\}_{\alpha \in A} \subseteq \tau$, then $\bigcup_{\alpha \in A} U_\alpha \in \tau$
3. $\tau$ is closed under finite intersections: if $U_1, U_2, \ldots, U_n \in \tau$, then $\bigcap_{i=1}^{n} U_i \in \tau$

The elements of $\tau$ are called **open sets**.

## Example: The Standard Topology on $\mathbb{R}$

The standard topology on $\mathbb{R}$ is generated by the basis of open intervals:

$$
\mathcal{B} = \{(a, b) \mid a, b \in \mathbb{R}, \, a < b\}
$$

Every open set in $\mathbb{R}$ can be written as a countable union of open intervals. This is a consequence of $\mathbb{Q}$ being dense in $\mathbb{R}$.

## Continuity in Topological Spaces

A function $f: (X, \tau_X) \to (Y, \tau_Y)$ is **continuous** if for every open set $V \in \tau_Y$, the preimage $f^{-1}(V) \in \tau_X$.

This generalizes the $\varepsilon$-$\delta$ definition from analysis. In fact, for metric spaces the two definitions coincide:

$$
\forall \varepsilon > 0, \, \exists \delta > 0 : d_X(x, x_0) < \delta \implies d_Y(f(x), f(x_0)) < \varepsilon
$$

## Computing Fundamental Groups with Python

Here's a simple implementation that computes the fundamental group of a wedge of circles using the van Kampen theorem:

```python
from dataclasses import dataclass
from typing import List

@dataclass
class FundamentalGroup:
    generators: List[str]
    relations: List[str]

    def __repr__(self):
        if not self.relations:
            return f"Free({', '.join(self.generators)})"
        rels = ', '.join(self.relations)
        return f"<{', '.join(self.generators)} | {rels}>"

def wedge_of_circles(n: int) -> FundamentalGroup:
    """π₁ of a wedge of n circles is the free group on n generators."""
    generators = [f"a_{i}" for i in range(1, n + 1)]
    return FundamentalGroup(generators=generators, relations=[])

# Example: π₁(S¹ ∨ S¹) = F₂
pi_1 = wedge_of_circles(2)
print(pi_1)  # Free(a_1, a_2)
```

## The Euler Characteristic

For a compact surface, the Euler characteristic $\chi$ is a topological invariant:

$$
\chi = V - E + F
$$

where $V$, $E$, $F$ are the number of vertices, edges, and faces in any triangulation. For the $n$-holed torus (genus $g$):

$$
\chi(\Sigma_g) = 2 - 2g
$$

## Implementing Simplicial Complexes in C++

```cpp
#include <vector>
#include <set>
#include <algorithm>

struct SimplicialComplex {
    std::vector<std::set<int>> simplices;

    void add_simplex(const std::set<int>& sigma) {
        // Add all faces of the simplex
        for (size_t k = 1; k <= sigma.size(); ++k) {
            std::vector<int> elems(sigma.begin(), sigma.end());
            std::vector<bool> selector(elems.size(), false);
            std::fill(selector.begin(), selector.begin() + k, true);
            do {
                std::set<int> face;
                for (size_t i = 0; i < elems.size(); ++i) {
                    if (selector[i]) face.insert(elems[i]);
                }
                simplices.push_back(face);
            } while (std::prev_permutation(selector.begin(), selector.end()));
        }
    }

    int euler_characteristic() const {
        int chi = 0;
        for (const auto& s : simplices) {
            chi += (s.size() % 2 == 1) ? 1 : -1;
        }
        return chi;
    }
};
```

## Key Takeaways

- Topological spaces generalize metric spaces by keeping only open-set structure
- Continuity is defined purely in terms of preimages of open sets
- The fundamental group $\pi_1(X, x_0)$ captures the "loop structure" of a space
- Euler characteristic is computable and invariant under homeomorphism
